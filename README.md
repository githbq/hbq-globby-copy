# globby-copy-promise

## 功能介绍

1. globby-copy-promise

## 安装

``` 
yarn add globby-copy-promise 
//or 
npm install globby-copy-promise 
```

## 使用   

``` 
await copy(['**/*.js','!**/node_modeuls/**/*'], '/home/temp')
```

## cli模式

``` js
npm i - g globby - copy - promise

globby - copy '**/*.js'
'!**/node_modeuls/**/*' - d = '/home/temp'
```
